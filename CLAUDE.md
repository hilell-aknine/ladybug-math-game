# משחק חשבון — החיפושית המופלאה 🐞

## מה זה
משחק חשבון לגילאי 5-6 בנושא הסדרה "החיפושית המופלאה" (Miraculous Ladybug).
הילד פותר תרגילי חשבון, מרוויח מטבעות, וקונה פריטים לבית/מאפייה של ליידיבאג.

## הפעלה
פתח `index.html` בדפדפן. לא צריך שרת. הכל בקובץ HTML אחד.

## מבנה
```
ladybug-math/
├── index.html              ← כל המשחק (HTML + CSS + JS inline)
├── CLAUDE.md               ← הקובץ הזה
├── generate-all.sh         ← סקריפט יצירת תמונות AI (Gemini)
├── generate-assets.py      ← סקריפט ישן (Pollinations, לא פעיל)
├── assets/images/          ← תמונות AI (נוצרות עם generate-all.sh)
│   ├── characters/         ← ladybug, cat-noir, tikki, ladybug-celebrate
│   ├── backgrounds/        ← welcome, bakery, shop, map
│   └── items/              ← 12 פריטי חנות
├── css/                    ← קבצי CSS מקוריים (לא בשימוש, הכל inline)
└── js/                     ← קבצי JS מקוריים (לא בשימוש, הכל inline)
```

## מסכים (5)
1. **מסך פתיחה** — שם הילד, כוכבים מהבהבים, דמות ליידיבאג
2. **מפה** — 4 תחנות תרגול + חנות + בית, רמה + XP bar
3. **תרגילים** — 10 שאלות בסבב, כפתורי מספרים חכמים (8 אופציות)
4. **חנות** — 12 פריטים, טאבים לפי קטגוריה, מודל אישור קנייה
5. **בית** — מאפייה עם 11 משבצות, tap-to-select → tap-to-place

## סוגי תרגילים
| סוג | טווח | נפתח אחרי |
|-----|------|-----------|
| ספירה | 1-10 | מיידי |
| חיבור | עד 10→20 | מיידי |
| חיסור | עד 10→20 | 10 חיבורים |
| גדול/קטן | עד 20 | 10 חיסורים |

## מערכות
- **מטבעות** — +1 לתשובה נכונה, +1 בונוס על רצף 3
- **רמות** — מתחיל🌱 → חוקר🔍 → גיבור⭐ → אלוף🏆 → מאסטר👑
- **חנות** — 12 פריטים (3-15 מטבעות), טאבים: הכל/מאפייה/רהיטים/קישוט
- **בית** — 11 משבצות (מטבח, דלפק, קיר, חלון, ספה, רצפה)
- **צלילים** — Web Audio API, בלי קבצי שמע חיצוניים
- **שמירה** — localStorage, שומר מצב אוטומטית

## תכונות UX
- כפתורי מספרים חכמים (רק 8 אופציות רלוונטיות)
- טיקי מסקוט — בועת דיבור עם עידוד, חגיגה על תשובה נכונה
- קונפטי + כוכבים על תשובה נכונה
- רצף (streak) עם בונוס
- חגיגה כשנפתח שלב חדש
- כפתור השתקה
- שמירה אוטומטית ב-localStorage
- Emoji fallback — המשחק עובד גם בלי תמונות AI

## יצירת תמונות AI
הסקריפט `generate-all.sh` משתמש ב-nano-banana-poster (Google Gemini) ליצירת 20 תמונות.
```bash
bash generate-all.sh
```
- הקוואטה מתאפסת יומית (free tier)
- מחכה 60 שניות בין כל תמונה
- התמונות נכנסות ל-`assets/images/` ומוצגות אוטומטית במשחק

## טכנולוגיה
- HTML/CSS/JS — קובץ יחיד, אין dependencies
- Web Audio API — צלילים פרוצדורליים
- localStorage — שמירת מצב
- Google Fonts — Rubik
- RTL — עברית מלאה
